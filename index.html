⚠️ ล้างข้อมูลทั้งหมด?

คุณต้องการล้างผลการแข่งขันและตารางคะแนนทั้งหมดหรือไม่? การกระทำนี้ไม่สามารถย้อนกลับได้

🔧 ตั้งค่า Firebase — ซิงค์ข้ามอุปกรณ์

ตั้งค่า Firebase เพื่อให้ข้อมูลแสดงเหมือนกันทุกเครื่อง (คอม, มือถือ, แท็บเล็ต) แบบ real-time ฟรี!

  1. 1
    เข้าไปที่ console.firebase.google.com แล้วสร้าง Project ใหม่ (ตั้งชื่ออะไรก็ได้ เช่น smash-x)
  2. 2
    ในเมนูซ้าย คลิก Build → Realtime Database แล้วกด "Create Database"
    เลือก location: Singapore แล้วเลือก "Start in test mode"
  3. 3
    กลับไปหน้า Project Overview (มุมบนซ้าย) → คลิกไอคอน </> (Web) เพื่อเพิ่ม Web App
    ตั้งชื่อ App แล้วกด Register app → จะเห็น firebaseConfig
  4. 4
    คัดลอกค่าจาก firebaseConfig มาวางในช่องด้านล่าง แล้วกด บันทึก
